# Tickd

A clean, minimal task management web application built to help users organize work, manage projects, and stay focused without distractions.

---

## 🚀 Live Demo

🌐 https://tickd-iota.vercel.app/

---

## 📂 GitHub Repository

🔗 https://github.com/kishorrjeyachandran/Tickd

---

## ✨ Features

- 🔐 User Authentication
- ➕ Add / Edit / Delete Tasks
- 📅 Due Date Management
- 📁 Project-based Task Organization
- ⚡ Priority System
- 📊 Analytics Dashboard
- 📱 Fully Responsive Mobile UI
- 🎨 Minimal & Modern Interface
- ☁️ Cloud Sync with Supabase

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion
- Lucide React

### Backend & Database
- Supabase
- PostgreSQL

- Vercel
